catena::common::Authorizer::Authorizer ( const std::string &  JWSToken)

Constructs a new Authorizer object by extracting client scopes from a JWS token. Authorizer expects the JWSToken to be valid. Authentication of the token is to be handled by the API gateway.

Parameters
JWSTokenThe JWS token to extract scopes from.
Exceptions
Throwsan catena::exception_with_status UNAUTHENTICATED if the authorizer fails to decoded the JWS Token.
