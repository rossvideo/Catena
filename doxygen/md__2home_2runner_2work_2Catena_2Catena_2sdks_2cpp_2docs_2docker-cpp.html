Using Docker With The Catena C++ SDK

Terminal

Requirements

  • Install Docker and add it to your system path
  • If on Linux, make sure to enable docker as non-root user

In the Catena/sdks/cpp/docker/ directory run ./dev-start.sh <absolute/path/to/Catena>

Note: if you get the error:

exit status 22: unpigz: abourt: zlib version less than 1.2.3

download the latest version of pigz.

VSCode Workspaces

Extensions Required

  • Docker additionally follow the Docker software install instructions listed in the extension if not already set up.
  • Dev Containers for integration with the Catena workspace.
  • Cmake Tools for build setup

Steps to run

Press F1 to open the command palette and select the option Dev Containers: Open worspace in Container, navigating to your Catena workspace file

Note: if you get the error:

exit status 22: unpigz: abourt: zlib version less than 1.2.3

download the latest version of pigz.

Ensure the C/C++ Extension pack is enabled in the container as well as any other extensions you might want.

Continue to sdks/cpp/build, emptying it and using

cmake -G Ninja -DCONNECTIONS=gRPC -DCATENA_MODEL=lite ..
