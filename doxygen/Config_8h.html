Parses environment variables and command line to initialize variables used for runtime configuration.

Parameters
argcCommand-line argument count
argvCommand-line arguments
serviceTypeThe type of service being ran, can be "REST" or "GRPC".
prefixPrefix used for relevant environment variables, include trailing underscore. Defaults to "CATENA_".
Returns
<bool exit, int code>. Exit states if program must be exited, either due to –help or an exception. Code denotes the reason for exit, with 0 being –help and 1 being an exception. The function will print the relevant material to the terminal before returning.
Examples
enum_decorator.cpp, factory.cpp, and path.cpp.
