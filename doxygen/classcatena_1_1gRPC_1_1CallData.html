std::string catena::gRPC::CallData::jwsToken_ ( ) const
inlineoverrideprotectedvirtual

Extracts the JWS Bearer token from the client's metadata.

Client metadata is included in the ServerContext. The bearer token can then be found under the "authorization" key.

Returns
The JWS Bearer token with the "Bearer " prefix removed.
Exceptions
catena::exception_with_statusif the token is not found.
