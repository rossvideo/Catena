Detailed Description

template<typename P, typename K, typename... Ms>
class catena::patterns::GenericFactory< P, K, Ms >

Factory template with which client classes can register a "Maker" function together with a unique key.

GenericFactories are Singletons.

Template Parameters
Pthe Product made by the factory.
Kthe key that uniquely identifies the product
Msparameters for the Maker function.
Since
1.0.0
