Detailed Description

template<typename T>
struct catena::patterns::Singleton< T >::Protector

This is a helper that allows derived classes to declare public constructors that take a Protector as an argument. Because the Protector is itself protected, it means that client code cannot call the derived class' public constructor because Protector is inaccessible.
This approach avoids the necessity of derived classes declaring Singleton as a friend so that it can access their (private) constructors.
