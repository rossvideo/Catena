◆ subParam() [1/2]

template<bool Threadsafe>
std::unique_ptr<ParamAccessor> catena::ParamAccessor::subParam ( const std::string &  fieldName)
inline

get accessor to sub-parameter matching fieldName

Parameters
fieldNamename of the sub-parameter
Returns
unique_ptr to ParamAccessor interface
Exceptions
catena::exception_with_statusif fieldName is not a sub-parameter
catena::exception_with_statusif fieldName is not a struct or variant type

Note that the value element of the returned sub-param references the appropriate part of the parent's value element. This is because the value element in the child param descriptor is not part of the active value of the larger object. If even present, it's used as the default value for the matching part of the parent's value object.

This method is threadsafe because it asserts the DeviceModel's mutex.

References BAD_STATUS, and value().
